# valkyria
This project was made as a study, and it's a Client consumer for TJ-SC's SOAP Web Service homologation environment.

It was built using TDD and Layer Architecture.

Can still be improved.
Feel free to make pull requests and sugestions.

To run this API:
  - Clone this repository to your PC.
  - `cd valkyria/`
  - `mvn clean install`
  - `mvn spring-boot:run`
